## Feature List
### User Features	
-	Simple sign up / login (optional)
-	Save favourite recipes and organise using the pin-to-top feature.
-	Sharing recipes with others will include formatted text and an image(if the sharing app allows).
  
### Core Features
-	Add, edit, and delete personal recipes.
-	Add cooking notes (for improvement).
-	Cooking steps tracking.
-	Filter recipes by dietary restrictions (gluten-free, vegan, nut-free, etc.).
-	Track cooking history (date, recipe cooked and viewed).
-	Online recipe search feature (Based on API TheMealDB)
  
### UI Elements (Simple and Straightforward)
- Orange theme(matchcing app logo)
-	Recipe cards (grid layout with images & titles).
-	Filters (window checkbox and chips for dietary preferences).
-	Recipe detail page (ingredients, steps, notes).
-	Bottom navigation bar for different screens (Home, Saved, Add, History, Profile).
